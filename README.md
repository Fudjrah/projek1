# Projek 1 - Sistem Manajemen Lagu (Laravel 12 CRUD)

Selamat datang di repositori Projek1 saya. Ini adalah aplikasi berbasis web yang dibangun menggunakan **Laravel 12** untuk memenuhi tugas pemograman website 2

## 🛠 Prasyarat Sistem
Sebelum menjalankan proyek ini, pastikan laptop Anda telah menginstal:
* **PHP >= 8.2**
* **Composer** (Versi terbaru)
* **Node.js & NPM** (Untuk menjalankan Vite)
* **XAMPP** (Untuk Apache & MySQL)

## Langkah Instalasi

Ikuti urutan perintah berikut di terminal Anda:

### 1. Clone Repositori & Instal Dependensi
```bash
git clone [https://github.com/Fudjrah/projek1.git](https://github.com/Fudjrah/projek1.git)
cd projek1

# Instal library PHP
composer install

# Instal & build aset CSS/JS (Vite)
npm install
npm run build
